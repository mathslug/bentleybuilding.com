# Bentley Building Company Website

This is a simple, static HTML/CSS website for Bentley Building Company, LLC.

## Structure

- `index.html` - The main and only HTML page
- `styles.css` - All styling for the website
- `img/` - Contains images used on the site
- Various favicon and icon files for different platforms

## Deployment

To deploy this website, just push changes to the main branch.

## Maintenance

To update the website:

- Edit `index.html` to change text content
- Modify `styles.css` to adjust styling
- Replace images in the `img/` folder as needed (maintain the same filenames)
